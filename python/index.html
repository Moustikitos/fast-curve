cSecp256k1#

cSecp256k1 is a ctypes binding that provides fast computation on SECP256K1 curve. ECDSA signature is about 140 times faster than pure python implementation, SCHNORR signature about 60 times faster.

Generate a random secp256k1 integer (in range [1..p]).

rfc6979_k#

rfc6979_k(msg, secret0, V=None)

Generate a deterministic rfc6967 integer.

HexPoint Objects#

class HexPoint(ctypes.Structure)

ctypes structure for secp256k1 curve point with xand y attributes as hex bytes.

Attributes:

  • x bytes - point absisse as hex bytes
  • y bytes - point ordinate as hex bytes
>>> import cSecp256k1 as cs
>>> G = cs.HexPoint(
...    b"79be667ef9dcbbac55a06295ce870b07029bfcdb2dce28d959f2815b16f81798",
...    b"483ada7726a3c4655da4fbfc0e1108a8fd17b448a68554199c47d08ffb10d4b8"
)
>>> G.x  # return x value as hex bytes
b'79be667ef9dcbbac55a06295ce870b07029bfcdb2dce28d959f2815b16f81798'
>>> G[0]  # return x value as integer
55066263022277343669578718895168534326250603453777594175500187360389116729\
240

Eliptic curve algebra is implented with python operator + and *.

>>> G * 2
<secp256k1 point:
    x:b'c6047f9441ed7d6d3045406e95c07cd85c778e4b8cef3ca7abac09b95c709ee5'
    y:b'1ae168fea63dc339a3c58419466ceaeef7f632653266d0e1236431a950cfe52a'
>
>>> G + G
<secp256k1 point:
    x:b'c6047f9441ed7d6d3045406e95c07cd85c778e4b8cef3ca7abac09b95c709ee5'
    y:b'1ae168fea63dc339a3c58419466ceaeef7f632653266d0e1236431a950cfe52a'
>

HexSig Objects#

class HexSig(ctypes.Structure)

ctypes structure for secp256k1 signature with rand s attributes as hex bytes.

Attributes:

  • r bytes - signature part 1 as hex bytes
  • s bytes - signature part 2 as hex bytes
